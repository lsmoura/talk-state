# React State Demo

Some demos to show. Start a webpack-dev-server with `pnpm run dev` and try one of these routes:

* `/hello-world`
* `/stateless-table`
* `/sort-table`
* `/cork-board`
* `/improved-board`

## Building

Execute `pnpm run build` and find all the needed files on `/dist`.
